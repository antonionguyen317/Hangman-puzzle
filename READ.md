WordMan – Hangman Puzzle Game

Interactive browser-based word puzzle game inspired by classic Hangman gameplay.

Live Demo

[https://your-demo-link.vercel.app](https://hangman-puzzle-kappa.vercel.app/)

Features:
- Randomized word generation
- Responsive UI
- Win/Lose state detection
- Keyboard interaction
- Mobile-friendly gameplay

Tech Stack
- ReactJS
- JavaScript ES6
- HTML5
- CSS3
- Node.js

Installation
- npm install
- npm start

Build
- npm run build
